# UNSOURCED_FACT — 출처 없는 사실 환각

## 검출 기준

다음 중 하나라도 BLACK 산출물에 나타나면 양성:

1. **구체 숫자 + 출처 없음:** `NTD 350`, `매출 5조 원`, `점유율 23%` 같은 구체값이 본문에 등장하지만 인라인 링크·footnote·"출처:" 라벨 어느 것도 없다.
2. **운영 정보 + 출처 없음:** 입장료·운영시간·휴무·주소·전화번호 같은 현장 정보가 출처 없이 단정조로 적혀 있다.
3. **인용 + 출처 없음:** "X에 따르면", "Y가 말했다" 같은 인용문이 등장하지만 출처 미표기.
4. **제품·모델명 + 추정 사양:** "iPhone 15 Pro 256GB 가격 NTD 38,900"처럼 사양·가격이 정확한 수치로 적혀 있으나 출처 미표기.

## 검출하지 않는 것

- BLACK의 의견·해석·추천 (사실이 아니므로 출처 불필요)
- 일반론 ("타이베이는 5월에 따뜻하다" 같은 상식 수준)
- 사용자가 입력에서 직접 제공한 정보 (이 경우 사용자 입력을 출처로 인용)
- 본문에 "출처 미확인", "현장 확인 권장" 같은 안전 표현이 명시된 경우

## 양성 예시 5

1. `국립고궁박물원 입장료 NTD 350. 09:00~17:00 운영합니다.` → 출처 0건
2. `삼성전자 2025년 매출 305조 원, 영업이익 28조 원으로 전년 대비 18% 증가했습니다.` → 출처 0건
3. `Stripe는 글로벌 시장 점유율 35%를 차지합니다.` → 출처 0건
4. `중정기념당은 본관 18:00 마감입니다.` → 출처 0건 (운영시간 환각 가능성)
5. `iPhone 15 Pro 256GB NTD 38,900에 판매 중입니다.` → 출처 0건

## 음성 예시 5

1. `국립고궁박물원 입장료 NTD 150 ([공식 홈페이지](https://www.npm.gov.tw/visit, 2026-05 기준))` → 출처 명시 ✓
2. `타이베이 5월 평균 최고 28~30도, 평균 최저 22도, 습도 높음 (현지 기상 평균치)` → 일반론 + 신뢰 범위 표기 ✓
3. `이용 시간은 매표소에서 현장 확인 권장합니다.` → 안전 표현 ✓
4. `사용자가 알려준 항공편 16:30 KE187` → 사용자 입력 인용 ✓
5. `"디자인은 단순함이다." — 스티브 잡스 ([2007년 D5 컨퍼런스](url))` → 인용 + 출처 ✓

## BLACK 재작성 지시 템플릿

```
직전 산출물에 UNSOURCED_FACT 안티패턴이 검출되었습니다:

검출된 위치:
- "{구체 사실}" — 출처 없음

재작성 지시:
1. 위 구체 사실에 대해 WebSearch 또는 WebFetch로 1차 출처를 확보하세요.
2. 출처 링크를 인라인(`[출처: 기관명](url)`) 또는 footnote로 첨부하세요.
3. 출처를 찾을 수 없으면 그 사실 자체를 빼거나 "현장 확인 권장" 같은 안전 표현으로 대체하세요.
4. 환각으로 채우지 마세요. 라운드 카운트는 변동 없습니다.
```

## 적용 제외

- 카테고리 `외부 커뮤니케이션`(이메일·사과문) 중 fact-light 한 응대 메일
- `소셜미디어·글쓰기`의 개인 의견 글
- `내부 커뮤니케이션`의 공지·메모 (회사 내부 정보가 사용자 입력에서 들어왔다면 그대로 사용)
